# Hopper

A browser-based Frogger remake. Cross roads, ride logs, reach home.

## Play

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Controls

- Arrow keys or WASD to hop
- SPACE to start/restart
- P to pause/unpause

## Gameplay

1. Start at the bottom
2. Cross the road - avoid cars and trucks
3. Cross the river - hop on logs (don't fall in!)
4. Reach one of the home slots at the top
5. Fill all 5 homes to complete the level

## Scoring

- 10 points per forward hop
- 50 points for reaching home
- 1000 points for completing a level
- Time bonus based on remaining time

## Tech Stack

- Next.js 14 + TypeScript
- HTML5 Canvas for rendering
- Tailwind CSS for UI chrome
